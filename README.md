# ai-code-setup

Personal AI coding skills and configurations for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skills

| Skill | Description |
|---|---|
| [plan-commits](skills/plan-commits/) | Analyze uncommitted changes and organize them into logical, granular Conventional Commits |

## Installation

Copy a skill into your Claude Code skills directory:

```bash
cp -r skills/plan-commits ~/.claude/skills/plan-commits
```

## Usage

In Claude Code, use the slash command:

```
/plan-commits
```

Or describe what you want naturally — the skill triggers when you mention splitting, organizing, or planning commits.
